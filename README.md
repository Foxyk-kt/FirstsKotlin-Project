Android Calculator

Простой калькулятор на Jetpack Compose с поддержкой базовых арифметических операций.

## Возможности

- Сложение, вычитание, умножение, деление
- Процент `%` — от предыдущего числа, если оно есть, иначе делит на 100
- Смена знака `+/-`
- Удаление последнего символа `<`
- Полный сброс `AC`
- Цепочки операций: `2 + 3 * 4 =`
- Скрытый debug-режим — тап по верхней полоске показывает внутреннее состояние (числа, операторы, ошибки)

## Стек

- Kotlin
- Jetpack Compose
- Material 3
- Min SDK: 21

## Запуск

Клонируй репозиторий, открой в Android Studio, запусти на эмуляторе или устройстве. Зависимостей нет.

## Известные ограничения

- Нет обработки деления на ноль
- История вычислений не сохраняется
